# Xbox Subscription Dashboard

## Project Overview
This project presents an Excel dashboard created to analyze subscription data related to Xbox services, including:

- Xbox Game Pass
- EA Play Season Pass
- Minecraft Season Pass

The dashboard provides a visual summary of revenue and subscription distribution, with an interactive filter by subscription type.

---

## Dataset
The dataset contains information about subscribers and their subscriptions.  
The main columns include:

- Subscriber ID
- Name
- Plan
- Start Date
- Auto Renewal
- Subscription Price
- Subscription Type (Monthly, Annual, Quarterly)
- EA Play Season Pass
- EA Play Season Pass Price
- Minecraft Season Pass
- Minecraft Season Pass Price
- Coupon Value
- Total Value

---

## Dashboard Features
The Excel dashboard includes the following elements:

- Total Revenue card
- EA Play Season Pass revenue
- Minecraft Season Pass revenue
- Xbox Game Pass subscription comparison (Yes vs No)
- Interactive filter using a slicer for Subscription Type

## How to Reproduce

1. Download the Excel file located in the **dashboard** folder.
2. Open the file using Microsoft Excel.
3. Go to the **Dashboard** sheet.
4. Use the slicer to filter by:
   - Monthly
   - Annual
   - Quarterly
